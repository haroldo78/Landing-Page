ReadMe Landing Page

These projects have been made for existing companies in the real estate market. With conversion goal focused on turning your traffic into leads and your leads into customers.

Using Standards: PHP, HTML, CSS, Javascript, Images with components: Jquery, Jquery Validate, css Reset. Using Microsoft Excel as a database

Can be used with a local served xampp, mamp or of your preference given start in Apache to run PHP file of name of folder site index.php
